Welcome to Adm1nAbuse

ℹ️ What is Adm1nAbuse?

Adm1nAbuse is a fun events server hosted on Minehut! Each round, a random player is selected to be the admin. Every game, your chance of being the admin increases, and will also increase if people like your event when you're the admin!

More info below:

👑 ADMIN

  • Each round, one random player gets to be the ADMIN for 15 minutes!
  • Use custom commands to host an awesome event!
  • Unlock unique items to host cooler events!
  • Or you can torture everyone for FUN!

🎮 PLAYERS

  • Players compete in the events made by the admin!
  • The players must win the event at any cost!
  • Stand on the Blue Wool to win!

🪙 Coins

  • Earn coins by winning events!
  • Buy custom items to use while you're the Admin!
  • Unlock new map presets to make better events!
  • Buy chat colors to show off your wealth!

🛜 How to play

  • IP: Adm1nAbuse.minehut.gg
  • Version: 1.21.1+
  • Or join through the Minehut lobby using /join Adm1nAbuse!
